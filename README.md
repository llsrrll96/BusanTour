## 4. 핵심 기능
- 회원 관련 페이지 : 시큐리티 적용한 로그인/로그아웃, 회원가입
- 관리자 페이지 : 회원리스트, 회원강제탈퇴, 회원권한부여
- 메인 홈 게시판 : 관광지 카드형 게시판, 구군별-테마별 찾기
- 여행 리뷰 게시판 : 리뷰 등록/수정/삭제/조회
- 참여방 게시판 : 여행 참여자 모집
- 마이 메뉴 : 내가 쓴 여행리뷰, 내가 쓴 참여 모집 글
